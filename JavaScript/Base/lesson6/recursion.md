Числа Фибоначчи тесно связаны с золотым сечением и множеством природных явлений вокруг нас.<br>

Напишите функцию fib(n), которая возвращает n-е число Фибоначчи. Пример работы:<br>

```javascript
function fib(n) { /* ваш код */ }

alert( fib(3) ); // 2
alert( fib(7) ); // 13
```
